# MicroData Tools

Ferramentas gratuitas para limpar, analisar e converter arquivos CSV e Excel.

## Objetivo

O projeto tem como objetivo ajudar pessoas, estudantes, freelancers e pequenos negócios a transformar planilhas bagunçadas em dados limpos, relatórios simples e arquivos prontos para uso.

## Ferramentas previstas

- Limpador de CSV/Excel
- Conversor de dados
- Analisador de planilhas
- Futuro modo para análise de vendas de maquininha

## Stack inicial

- Python
- FastAPI
- Pandas
- OpenPyXL
- Pydantic
